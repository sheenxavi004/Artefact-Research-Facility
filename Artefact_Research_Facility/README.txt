The Java project is available at https://github.com/sheenxavi004/Artefact-Research-Facility
1. Initially extract all the files from the repository provided in the above link.
2. Then setup the project requirements as given in DEPENDENCY.txt .
3. Now using terminal, move to the dist folder of Artefact_Research_Facility
using the cd command :

$ cd Artefact_Research_Facility/dist

4. In-order to run the application execute the following command :

$ java -jar Artefact_Research_Facility.jar

5. The application is now ready for use.
